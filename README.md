## Guardian of Sanctuary

VR/AR/게임제작기초 수업의 기말 프로젝트로 제출한 게임, "Guardian of Sanctuary"입니다.

-   제작 기간: 2023년 11월 26일 ~ 12월 8일
-   장르: 1인칭 3D FPS 디펜스 게임
-   플랫폼: iOS
-   테스트 환경: iPhone 13, iOS 16.1
